System Requirements



The following are the System Requirements for DigiOz Guestbook:


  • PHP Version 5.0.0 or higher
  • A Web Server with PHP Support such as Apache Web Server or Windows IIS version 5 or higher.
  • GD Support for Simple Image Verification (Optional)
  • A Recaptcha Public and Private Key, if you intend on using Recaptcha (Optional). You can obtain a key for free at http://www.google.com/recaptcha.
  • Access to the Web Directory through FTP.
  • The permission to change the file permission on the Web Server, in order to set the Read, Write and Execute Permissions of various script files.
